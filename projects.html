Spotify-Linked Side by Side Lyrics Translator

A long standing goal of mine has been to improve my Spanish, which led to my enamoration with Latin music. While the music alone has proven to be a helpful tool for bettering my comprehension and speaking, the meaning behind many of my favorite lyrics still escapes me. So, I'm working on using the Spotify API to build a web app that finds the lyrics of whatever song I may be listening to, and uses AI to translate them to be displayed alongside the original lyrics, while synchronized to my Spotify session.


Built with Python, HTML, and CSS, using Flask, as well as the Spotify and OpenAI APIs.
